# Chasqui II project

This is the project for Chasqui II.

## Commands

For these project you will need [Nodejs >= 20](https://nodejs.org/en) and [pnpm >= 9](https://pnpm.io/installation).

## Commands


| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `pnpm install`             | Installs dependencies                            |
| `pnpm run dev`             | Starts local dev server at `localhost:5173`      |
